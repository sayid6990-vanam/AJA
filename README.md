# AHAD JUS DUX — iOS Liquid Glass Theme Background Update

This version uses the uploaded city images as full-site blurred background scenes:

- `dark-bg.png` for dark mode
- `light-bg.png` for light mode

## Upload structure

Upload all files directly into the root of your GitHub repository:

```text
index.html
style.css
script.js
logo.png
intro.mp4
dark-bg.png
light-bg.png
README.md
```

Do not place the files inside a `public` folder for this static Netlify version.

## What is included

- iOS-style liquid glass UI
- Dark/light theme toggle
- The uploaded city images used as blurred background layers
- Smooth reveal animations
- Floating glass navbar
- Scroll progress line
- Mobile glass menu
- Intro video retained
- Logo retained
- Contact form retained using mailto

## Netlify update steps

1. Replace the existing files in the repository root.
2. Commit the changes.
3. Wait for Netlify to redeploy.
4. Hard refresh the site:
   - Windows: `Ctrl + F5`
   - Mac: `Command + Shift + R`
